Network Architecture Project

This is an upcoming academic project focused on designing a robust and secure network architecture for a large enterprise.

The goal is to simulate a real-world scenario where we need to connect multiple branch offices to a headquarters while ensuring redundancy and security.

Key technologies involved:

  • Routing Protocols: Implementation of OSPF for internal routing and BGP for external connectivity.
  • VLANs & STP: Segmentation of the local network and loop prevention using Spanning Tree Protocol.
  • Security: Configuration of Firewalls and ACLs (Access Control Lists) to filter traffic.
