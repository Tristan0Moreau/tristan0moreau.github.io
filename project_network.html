Network Architecture Project

In this group project, we are designing a complete network architecture acting as an Autonomous System (AS). Our goal is to connect our AS to other groups.

The scenario involves external clients who need to connect securely to the company's internal network via a VPN infrastructure.

Key services and features:

  • Services: Implementation of VoIP for communication and FTP for file transfer.
  • Authentication: Secure identification system (username/password) for accessing company resources.
  • Routing: Management of the AS with BGP and internal routing.
  • Web page : Creation of a web page in the company network and a DNS service to access it.
