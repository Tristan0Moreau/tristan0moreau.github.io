Video Game Development

Combining my passion for gaming and my programming skills, I embarked on a project to create a video game from scratch with a friend.

We are currently in the design and prototyping phase. We are using modern game engines (like Unity or Godot) and coding the logic in C# / C++. This project allows me to explore various aspects of software engineering:

  • Game Physics: Implementing collision detection and movement mechanics.
  • Artificial Intelligence: Creating behaviors for non-player characters (NPCs).
  • Graphics & UI: Designing the user interface and integrating assets.

It is a challenging but incredibly rewarding project that pushes me to learn new technologies and solve complex algorithmic problems.
